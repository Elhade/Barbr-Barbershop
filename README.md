# BARB'R Barbershop

Site vitrine one-page pour le barber shop BARB'R à Marseille 13001.

## Description

Landing page responsive en HTML/CSS qui présente :
- l'ambiance du salon
- la galerie photo
- le concept "Le Vibe"
- la réservation via Instagram
- les informations de contact et l'adresse

Le design mise sur un style sombre, des accents orange et une navigation simple.

## Fichiers

- `index.html` : page principale
- `images/` : visuels et photos utilisées pour la section galerie et l'ambiance
- `netlify.toml` : configuration Netlify (déploiement statique)

## Aperçu local

Ouvre `index.html` dans ton navigateur pour voir le rendu. Tu peux aussi utiliser un serveur local simple si nécessaire :

```bash
python3 -m http.server 8000
```

puis visite `http://localhost:8000`

## Points forts

- accueil full-screen avec hero visuel
- section "L'Espace" avec description et images
- galerie photo en masonry
- cartes d'ambiance pour le vibe du salon
- CTA de réservation vers Instagram
- contact et horaires

## Remarques

- la réservation est gérée via Instagram : `https://instagram.com/barbr._13`
- le salon se situe au `1A Rue Chateauredon, 13001 Marseille`
- l'expérience est pensée pour le style street, la culture urbaine et le confort des clients
